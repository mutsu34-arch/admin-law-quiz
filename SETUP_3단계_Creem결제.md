## 테스트 방법

1. **패키지 설치** (아직 안 했다면)
   ```bash
   npm install
   ```

2. **서버 실행**
   ```bash
   npm run dev
   ```

3. **로그인** 후 **요금제 보기** 클릭

4. **구독하기** 버튼 클릭 → Creem 테스트 결제 페이지로 이동
