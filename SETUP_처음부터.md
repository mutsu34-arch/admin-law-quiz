# 행정법 퀴즈 앱 - 처음부터 설정하기

> 코딩 초보자를 위한 단계별 가이드입니다.

---

## 📌 이게 뭔가요?

- **Next.js** = 웹사이트를 만드는 도구 (React 기반)
- **npm** = 필요한 부품(패키지)을 설치하는 도구
- **터미널** = 컴퓨터에게 명령을 입력하는 창

---

## 1단계: 앱이 실행되는지 확인하기

### 1-1. 터미널 열기
- Cursor 왼쪽 하단 **터미널** 탭 클릭
- 또는 `Ctrl + `` (백틱) 누르기

### 1-2. 프로젝트 폴더로 이동
```
cd "c:\Users\mutsu\Desktop\행정법큐"
```
(한글 경로 때문에 안 되면, Cursor에서 이 폴더를 열어둔 상태에서 터미널을 열면 자동으로 이 폴더에 있습니다)

### 1-3. 부품 설치
```
npm install
```
(처음엔 1~2분 걸릴 수 있어요)

### 1-4. 앱 실행
```
npm run dev
```

### 1-5. 브라우저에서 확인
- `http://localhost:3000` 주소로 접속
- "행정법 퀴즈에 오신 것을 환영합니다!" 가 보이면 **성공** ✅

> ⚠️ **처음부터 다시 시작했습니다.** Firebase, Creem 등은 모두 제거했어요.
> 1단계가 잘 되면, 그다음에 구글 로그인 → Creem 결제 순서로 하나씩 추가할게요.

## ❓ 문제가 생겼을 때

### "npm을 찾을 수 없습니다"
→ Node.js가 설치되지 않은 것입니다. https://nodejs.org 에서 LTS 버전을 설치하세요.

### "포트 3000이 이미 사용 중"
→ 다른 프로그램이 3000번 포트를 쓰고 있습니다. 터미널에서 `Ctrl+C`로 앱을 끄고 다시 시도하세요.

### 한글 경로 문제
→ 프로젝트 폴더 이름을 `행정법큐` 대신 `admin-law-quiz` 같은 영문으로 바꿔보세요.
